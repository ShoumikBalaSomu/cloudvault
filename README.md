## 📖 Overview
**CloudVault** is a robust and secure personal cloud storage solution designed for privacy-conscious users. It features end-to-end encryption, multi-user support, and a streamlined administrative panel for managing datasets and users.

## ✨ Key Features
- 🔒 **End-to-End Encryption**: Securely store your files with advanced cryptographic modules.
- 👥 **Multi-User Support**: Manage different user roles and permissions easily.
- 🛠️ **Admin Panel**: A clean dashboard to monitor system health and user activities.
- 🚀 **High Performance**: Optimized for fast file retrieval and streaming.
- 📱 **Responsive Design**: Accessible from any device.

## 🛠️ Tech Stack
- **Backend**: Python (Flask)
- **Database**: SQLite/SQLAlchemy
- **Frontend**: HTML5, CSS3 (Modern Aesthetics)
- **Security**: Advanced Cryptography Modules

## 🚀 Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/shoumikbalasomu/cloudvault.git
   cd cloudvault
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Initialize the database**:
   ```bash
   python database.py
   python setup_admin.py
   ```

4. **Run the application**:
   ```bash
   bash run.sh
   ```
